Instructions for using app_settings.json

Open it in a text editor.

Values may be changed in this file to modify load-path, save-path, and filename 
without a GUI. run_app_headless should be set to true, because the GUI is not 
ready. Choose only 0ne of the assets to change to true. You can change asset 
path values to a different location if you wish to analyze a zip saved in a 
custom location. So, if you wish to analyze an older release of the Assets.zip 
file, you can modify the pregenerated path. All paths must use forward-slashes: 
( / ). All paths must end with a forward-slash. Filenames can be whatever you 
like.
